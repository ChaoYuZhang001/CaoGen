# CaoGen 终极目标 · 需求规格

> 本文把终极产品目标分解为**可验收需求**。每条:`编号 · 需求 · 验收标准 · 状态`。
> 状态取值:✅ 已验证(主控亲手 E2E/自动化闭环)· 🟡 链路通但有已知缺陷或待桌面复验 · 🧩 后端就位待接线/待外部依赖 · ⬜ 未实现。
> 愿景见 [ROADMAP.md](./ROADMAP.md);设计见 [DESIGN-V2.md](./DESIGN-V2.md)。措辞原则:"世界第一/中国首创"是**验收方向**,非当前事实宣称。

## 0 · 北极星(总验收)

- **N1** 任一主流 Agent(Codex/Claude Code/Gemini CLI/Cursor/Cline/Aider/通义灵码…)深度用户,**30 分钟内**在 CaoGen 跑通日常主链路。验收:迁移向导导入其 rules/MCP/技能后,该用户无需查文档完成"建会话→@文件→改代码→审 diff→提交"。状态:🟡(迁移向导 D11 在;30 分钟未做用户实测)
- **N2** **资产零丢失**:导入不破坏源工具配置,MCP/技能/rules 在 CaoGen 内等效可用。验收:导入后源工具仍正常,CaoGen 内技能/MCP 可调用。状态:🧩(继承 ~/.claude;跨工具导入矩阵部分)
- **N3** **核心工作流不倒退**:高频动作(@ / / / 图片 / 回退)在 CaoGen 至少与源工具持平。验收:逐项对照 DESIGN-V2 迁移矩阵无功能缺口。状态:🟡

## 支柱一 · 多厂商模型接入

- **P1.1** 可配置任意 Anthropic 兼容厂商(baseUrl+密钥+模型+自定义头)。验收:新增 Provider 后能建会话并成功对话。状态:✅
- **P1.2** 密钥加密落盘,永不出主进程、不入渲染层。验收:providers.json 无明文;渲染层拿不到 token。状态:✅
- **P1.3** 国产模型一等公民:DeepSeek/Kimi/智谱 GLM 官方端点内置直连预设,无须网关。验收:选国产预设即可对话。状态:🟡(网关可达;官方直连预设需核对)
- **P1.4** API key 自动拉取模型列表。验收:填 key 点"获取模型"返回真实列表。状态:✅
- **P1.5** OpenAI 原生引擎(Responses API)直连。验收:选 openai 引擎能对话。状态:🧩(已注册,本机未配 key 未 E2E)
- **P1.6** Codex/Gemini CLI 原生引擎。验收:装 CLI 后选对应引擎能对话。状态:🧩(适配器就位,本机未装 CLI)

## 支柱二 · 双模式智能调度

- **P2.1** 手动指定模型运行。验收:选定模型即以该模型跑。状态:✅
- **P2.2** 自动调度:按任务复杂度×策略(质量/成本/均衡)选模型。验收:同一策略下简单/复杂任务路由到不同档位模型,理由可见。状态:✅
- **P2.3** Provider 健康度追踪(成功/失败/延迟/连续失败)。验收:失败后健康度下降并在 UI 可见。状态:✅
- **P2.4** 跨厂商故障切换:厂商挂掉自动切健康厂商 resume 重试,任务不中断,聊天流透明标注。验收:主厂商失败时自动换家继续。状态:✅
- **P2.5** 预算闸门:超出每会话/每 Routine 预算上限时拦截。验收:超预算轮次被拒并提示。状态:⬜(budgetUsd 有字段无闸门)

## 支柱三 · 写实 3D 办公区

- **P3.1** 每会话一个工位,状态驱动小人动画(idle/working/awaiting/error)。验收:会话状态变化时对应工位动画切换。状态:✅
- **P3.2** 写实场景:房间/墙/落地窗/吊顶 + 全套家具(桌椅/显示器/书架/沙发/会议桌/白板/机架/茶水角/盆栽)。验收:进办公区可见完整房间。状态:✅
- **P3.3** Bloom 辉光 + 接触阴影 + 浮尘 + 昼夜主题自适应。验收:夜/日主题下观感协调。状态:✅
- **P3.4** 厂商主题小人(身体色=厂商品牌色)。验收:不同厂商会话小人配色不同。状态:✅
- **P3.5** 点工位聚焦会话 + 列表/办公区双视图切换。验收:点工位切回该会话聊天。状态:✅
- **P3.6** 多 Agent 协作可视化:工位间飞行消息包 + 小人离席去会议桌交流。验收:并行会话时可见协作动画。状态:🟡(氛围动画✅;为派生非真编排数据)
- **P3.7** 工位浮动成本/token 气泡。验收:工位上方显示该会话开销。状态:✅

## 支柱四 · 迁移级深度工作流

- **P4.1** `@` 文件引用:补全 + 内容注入上下文。验收:@文件后 agent 能读到其内容。状态:✅
- **P4.2** 图片输入:粘贴/拖拽 → base64 注入。验收:贴图后 agent 能描述图片。状态:✅
- **P4.3** 斜杠命令:`/` 唤起面板执行动作。验收:/ 列命令并可执行。状态:✅(本地面板命令)
- **P4.4** 检查点回溯 code 模式:回退文件到某轮之前。验收:dryRun 预览+应用真回退文件。状态:✅
- **P4.5** 检查点 chat/both 模式:回退对话上下文。验收:回退后 agent 不再记得被删轮次。状态:🟡(截断 CaoGen 转录;未回退 SDK 上下文)
- **P4.6** Git worktree 隔离:每 Agent 默认独立 worktree,互不污染;patch 导出。验收:worktree 内改动不影响原仓库,patch 含全部改动。状态:✅
- **P4.7** worktree 合并审查:子 worktree 改动审查后合并回主干。验收:合并前可审查、可应用。状态:🟡(WorktreePanel 已接合并摘要/patch 预览/apply-check/应用到主工作区;worktree smoke/typecheck/build/integration 通过,PR 与冲突三栏待补)
- **P4.8** 内置终端:交互式 shell。验收:命令写入有回显。状态:✅
- **P4.9** 内置文件编辑器:列/读/存项目文件。验收:改文件能保存到磁盘。状态:✅
- **P4.10** Diff 查看器:工作区改动可视化。验收:改动以红绿差异呈现。状态:🟡(只读,无逐块 accept/reject)
- **P4.11** 产物预览:HTML/文本/MD/CSV/JSON。验收:对应文件可预览。状态:✅(文本类)
- **P4.12** 图片/PDF 预览。验收:图片显示、PDF 可读。状态:✅(preparePreview 返回受限 dataUrl;图片与 PDF 内嵌预览已通过 preview smoke/typecheck/build/page-operation smoke)
- **P4.13** 内置浏览器 + 网页批注喂 Agent。验收:框选网页元素+指令交给 agent。状态:🟡(链路通;切面板原生视图遮挡已通过 page-operation smoke 修复验证;批注截图/DOM 圈选仍待补)
- **P4.14** 拖拽分屏布局。验收:面板可拖拽重排。状态:🟡(工作台第一版)
