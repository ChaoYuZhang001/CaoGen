# Reference office robot

`reference-office-robot.glb` is generated by
`scripts/generate-reference-robot-glb.mjs` from the official Unitree G1 rev 1.0
assets vendored in `third_party/unitree-g1-rev1`.

- Upstream repository: https://github.com/unitreerobotics/unitree_rl_gym
- Pinned upstream commit: `276801e46c5d433564f24658bac64f254b7d2d4b`
- Source license: BSD-3-Clause, retained in
  `third_party/unitree-g1-rev1/LICENSE`

The GLB contract follows exact official-source mesh names rather than the old
hand-built palm-shell and helmet-part names:

- `official_head_link` binds the `head_link` mesh.
- `official_torso_link` or `official_torso_link_23dof_rev_1_0` binds the
  `torso_link_23dof_rev_1_0` mesh.
- The current export uses `official_left_rubber_hand` and
  `official_right_rubber_hand`. A full 23-DOF export may use the exact upstream
  names `official_left_wrist_roll_rubber_hand` and
  `official_right_wrist_roll_rubber_hand` instead.
- Runtime animation hooks bind the exact roots `helmet_head`, `left_arm`,
  `right_arm`, `left_leg`, and `right_leg`.

The official `logo_link` mesh is not baked into the office GLB. Provider/vendor
logos are rendered at runtime by `ProviderLogoBadge` at
`provider_nameplate_mount`.
